## 💡 常见问题

### Q: `npm start` 执行后很久没弹出窗口？
A: 首次启动需要下载 Electron 二进制文件（约 100MB）。请检查终端是否有下载进度，耐心等待 1-5 分钟。

### Q: 如何确认是桌面窗口而不是浏览器？
A: 检查窗口标题栏：
- ✅ 桌面窗口：标题「剧本杀司机接单台」+ 最小化/最大化/关闭按钮
- ❌ 浏览器：有地址栏、标签页、书签栏

### Q: 弹出窗口显示空白或加载失败？
A: 请按 Ctrl+C 停止终端，然后重新执行 `npm start`。如果持续失败，可以先运行 `npm run dev` 确认浏览器模式正常。

### Q: 数据不对，想重新开始？
A: 点击页面右上角的「重置数据」按钮，或在浏览器/窗口控制台执行：
```javascript
localStorage.clear();
location.reload();
```

### Q: Electron 报错说找不到模块？
A: 请重新安装依赖：
```bash
cd d:\trae-bz\TraeProjects\2125
npm install
npm run electron:check
```
